About Genie Market

Genie Market helps manage structured information related to short-form content, product-focused media assets, and performance review workflows.

The service supports internal analysis of content outputs and simplifies how selected data is transformed into operational datasets for reporting and review.

What the service does

  • Collects limited authorized account and content metadata.
  • Structures content records into internal workflow-ready formats.
  • Supports product validation and content performance review.
  • Helps connect approved data sources to internal automation systems.

TikTok Integration

Genie Market may use TikTok Login and approved TikTok API access to retrieve limited authorized information after user consent.

Requested scopes are limited to the minimum needed for the integration, including basic profile access and video list access when approved for the app.

Authorized access only

  • No access is attempted without explicit user authorization.
  • No credentials are requested directly from users for TikTok login.
  • Only approved scopes and authorized API flows are used.
  • Access can be revoked by the user through the relevant platform controls.

Data Usage

Data is used only for legitimate service functions such as content analytics, workflow automation, internal reporting, product validation review, and record organization.

Genie Market does not sell personal data and does not use authorized data for unrelated advertising brokerage or unauthorized redistribution.

Security and retention

  • Access is limited to service operation and maintenance needs.
  • Sensitive integration values are stored separately from public website files.
  • Data is retained only as long as required for operational, legal, or security purposes.
  • Reasonable administrative and technical safeguards are applied.
